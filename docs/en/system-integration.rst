Supported Communication Protocols
---------------------------------

Our network products support the following protocols for system integration:

- **SNMP** (Simple Network Management Protocol)
- **MQTT** (Message Queuing Telemetry Transport)
- **BACnet** (Building Automation and Control Networks)
- **Modbus TCP** (Industrial Communication Protocol)
- **IFTTT** (If This Then That automation platform)
- **Email Notifications**
- **Home Assistant** (Open source home automation platform)

For detailed technical documentation on these protocols, visit:

`https://www.monigear.com/download/techdoc/ <https://www.monigear.com/download/techdoc/>`_

Cloud Platform Integration
--------------------------

The WTHM series devices are compatible with mainstream IoT platforms:

- AWS IoT Cloud
- Azure IoT Cloud
- Tuya IoT Cloud
