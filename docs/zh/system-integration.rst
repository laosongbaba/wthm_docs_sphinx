支持的通信协议
--------------

我们的网络产品支持以下协议进行系统集成：

- **SNMP** (简单网络管理协议)
- **MQTT** (消息队列遥测传输)
- **BACnet** (楼宇自动化和控制网络)
- **Modbus TCP** (工业通信协议)
- **IFTTT** (If This Then That 自动化平台)
- **电子邮件通知**
- **Home Assistant** (开源家庭自动化平台)

有关这些协议的详细技术文档，请访问：

`https://www.monigear.com/download/techdoc/ <https://www.monigear.com/download/techdoc/>`_
