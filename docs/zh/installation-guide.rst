安装说明
==========

产品支持两种安装方式：

方式一：安装背板支架+螺丝固定方式
----------------------------------

这种方式是通过安装背板支架并使用螺丝将设备固定在墙面或平面上。

.. figure:: /_static/images/wthm-install-m1.png
   :alt: 安装背板支架+螺丝固定方式示意图
   :width: 600px

   图：安装背板支架+螺丝固定方式

方式二：挂件安装方式
--------------------

这种方式是通过挂件（为选配件）将设备悬挂在墙面或其他支撑结构上。

.. figure:: /_static/images/wthm-install-m2.png
   :alt: 挂件安装方式示意图
   :width: 450px

   图：挂件安装方式
